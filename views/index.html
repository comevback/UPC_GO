API Documentation

Basic Endpoint

GET / ---- This endpoint returns a documentation for the API Server.

File Endpoints

Upload File

POST /files ---- This endpoint allows a user to upload a file.

curl -X POST http://localhost:port/files -F "file=@/path/to/file"

Get Files

GET /files ---- This endpoint lists all files uploaded by a user.

curl -X GET http://localhost:port/files

Delete File

DELETE /files/:filename ---- This endpoint deletes a specific file.

curl -X DELETE http://localhost:port/files/{:fileId}

Generate Docker Image

POST /api/files/{filename} ---- This endpoint accepts a .zip file upload, unzips it, and generates a Docker image from the contents.

curl -X POST http://localhost:port/api/files/{:filename} -F "file=@{path_to_your_zip_file}"

Image Endpoints

Upload Image

POST /images ---- This endpoint allows a user to upload an image.

curl -X POST http://localhost:port/images -F "image=@/path/to/image"

Get Images

GET /images ---- This endpoint retrieves all images uploaded by a user.

curl -X GET http://localhost:port/images

Delete Image

DELETE /images/:imageName ---- This endpoint deletes a specific image.

curl -X DELETE http://localhost:port/images/{:imageId}

Result Endpoints

Get Results

GET /results ---- This endpoint retrieves all results for a user or a task.

curl -X GET http://localhost:port/results

Delete Result

DELETE /results/:filename ---- This endpoint deletes a specific result.

curl -X DELETE http://localhost:port/results/{:resultId}
